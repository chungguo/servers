# [WIP] 基于[Nest.js](https://nestjs.com)的微信公众号服务

敏感信息存放于根目录 .env 文件中：

```
# 默认服务端口 10000
PORT=10000

# 微信公众号 token
MP_TOKEN=

# 微信公众号 appid
MP_APPID=

# 微信公众号 appsecret
MP_APP_SECRET=

# 微信公众号 encodingAESKey
MP_ENCODING_AES_KEY=

# 微信公众号默认回复
MP_DEFAULT_TEXT_RESP=

# MonggoDB
MONGODB_URL=
```

# 开发指引
## 本地启动
```
npm run start:dev
```
